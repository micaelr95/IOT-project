# Para executar
* Instalar o Python 3
* Instalar o Pyvenv
    * $ pip install virtualenv
* Fazer clone do repositório
* Criar uma pasta do Pyvenv na pasta do projeto
    * $ virtualenv .
* Ativar o Pyvenv
    * $ Scripts/activate
* Instalar os requirements
    * $ pip install -r requirements.txt
* Criar a variável de ambiente
    * set FLASK_APP=index.py
* Correr servidor
    * flask run
